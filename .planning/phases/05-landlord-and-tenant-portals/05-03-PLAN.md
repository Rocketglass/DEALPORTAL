**getTenantApplications(contactId: string)**
- Query `applications` table where `contact_id = contactId`
- Join: property (properties), unit (units), documents (application_documents)
- For each application, also fetch the associated LOI (if exists) by querying `lois` where `application_id` matches. Select: id, status, sent_at, agreed_at, and loi_sections (id, section_key, status) for section-level LOI status
- For each application, also fetch the associated lease (if exists) by querying `leases` where `application_id` matches. Select: id, status, docusign_status, signed_date
- Return type: `TenantApplicationWithDeal[]` -- define this interface:

